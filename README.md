# 划词翻译

一个简洁优雅的浏览器划词翻译用户脚本，支持选中英文文本后快速翻译成中文。

## ✨ 功能特性

- 🎯 **智能识别**：自动检测选中的英文文本，显示翻译图标
- 🚀 **快速翻译**：使用 Google Translate API，翻译速度快
- 🎨 **现代界面**：基于 React + Tailwind CSS 构建的现代化 UI
- 📋 **一键复制**：支持快速复制翻译结果
- 🔄 **拖拽移动**：翻译面板支持拖拽调整位置
- 👁️ **原文显示**：可选择显示或隐藏原始文本
- 🌐 **全站支持**：适用于所有网站
